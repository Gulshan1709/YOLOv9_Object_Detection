# YOLOv9 Real-Time Object Detection

This project implements a real-time object detection system using YOLOv9, Python, and OpenCV.

## Features
- Real-time webcam object detection
- Bounding boxes with class labels and confidence
- GPU acceleration (if available)
- Optimized for performance

## Requirements
- Python 3.9+
- OpenCV
- PyTorch
- Ultralytics

## How to Run
```bash
pip install torch ultralytics opencv-python
python yolov9_object_detection.py
